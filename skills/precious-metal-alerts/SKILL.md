# 贵金属价格提醒技能

## 功能概述

获取黄金（GOLD）和白银（SILVER）的实时价格，生成简洁的价格播报，支持定时提醒。

## 数据来源

按优先级尝试以下数据源：

1. **金十数据 (jin10.com)** - 国际贵金属期货
   - 黄金：COMEX黄金（GC=F）
   - 白银：COMEX白银（SI=F）

2. **东方财富贵金属频道** - 国内贵金属
   - 伦敦金、伦敦银

3. **新浪财经** - 备用数据源

4. **手动输入** - 如果自动获取失败，允许用户手动提供价格

## 触发方式

### 手动触发
用户主动查询时，直接获取最新价格并播报。

### 定时触发（Cron）
设置每天早上 9:00 自动执行提醒。

## 输出格式

### 价格播报模板
```
📈 贵金属价格播报 - [日期]

🪙 黄金 (XAU/USD)
  当前价格: $X,XXX.XX/盎司
  涨跌: +X.XX% / -X.XX%

🥈 白银 (XAG/USD)
  当前价格: $XX.XX/盎司
  涨跌: +X.XX% / -X.XX%

---
数据来源: [数据源名称]
```

## 执行步骤

### Step 1: 尝试获取数据
按优先级尝试获取黄金和白银价格。

### Step 2: 解析数据
从页面中提取：
- 当前价格 (Current Price)
- 涨跌幅 (Change %)

### Step 3: 生成播报
按模板格式生成价格播报，使用 emoji 装饰（🪙黄金、🥈白银）

### Step 4: 发送提醒
使用 message 工具将播报发送到用户当前会话或指定渠道。

## 错误处理

- 如果所有数据源都无法获取价格，输出友好提示并提供最近的市场综述
- 可以询问用户是否手动输入价格

## 定时提醒设置

如需设置每天9点自动提醒，执行以下命令配置 Cron：

```bash
openclaw cron add --skill precious-metal-alerts --schedule "0 9 * * *" --channel webchat
```

解释：`0 9 * * *` = 每天早上9点0分执行

## 依赖工具

- browser: 用于访问金融网站获取价格数据
- message: 用于发送提醒消息
